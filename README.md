# Business Regression Analysis – Dataset Understanding

## Problem Summary

The objective of this analysis is to identify the factors that are most strongly associated with **monthly sales performance** across retail stores. The insights will help leadership make data-driven decisions regarding marketing investment, inventory management, discount strategy, staffing, and store prioritization.

---

# 1. Dependent Variable (Target Variable)

The variable to be predicted in the regression model is:

| Variable | Description |
|------------|--------------------------------|
| **monthly_sales** | Total monthly sales generated by each store |

This will serve as the dependent (response) variable.

---

# 2. Potential Independent Variables (Predictor Variables)

The following variables can potentially explain variations in monthly sales:

| Variable | Type | Reason |
|-------------------------------|------------|--------------------------------------------|
| marketing_spend | Numerical | Marketing investment may increase sales |
| footfall | Numerical | More customers generally lead to higher sales |
| avg_discount_pct | Numerical | Discounts may influence purchase behavior |
| staff_count | Numerical | More staff may improve customer service and sales |
| inventory_availability_pct | Numerical | Better stock availability can increase sales |
| competitor_distance_km | Numerical | Stores farther from competitors may perform better |
| holiday_flag | Numerical (Binary) | Holidays may significantly impact sales |
| customer_rating | Numerical | Higher customer satisfaction may improve sales |
| region | Categorical | Sales may vary across geographical regions |
| store_type | Categorical | Different store formats may perform differently |
| month | Date/Time | Can capture seasonality after transformation |

---

# 3. Numerical Variables

The dataset contains the following numerical variables:

- marketing_spend
- footfall
- avg_discount_pct
- staff_count
- inventory_availability_pct
- competitor_distance_km
- holiday_flag (binary: 0/1)
- customer_rating
- monthly_sales
- monthly_profit

---

# 4. Categorical Variables

The following variables are categorical:

- store_id
- region
- store_type

These variables will require encoding before being used in regression models.

---

# 5. Variables That May Need Cleaning or Transformation

### month
- Stored as a date.
- Can be transformed into:
  - Month number
  - Quarter
  - Seasonal indicators
- Date values cannot be directly used in linear regression.

### region
- Requires one-hot (dummy) encoding.

### store_type
- Requires one-hot (dummy) encoding.

### competitor_distance_km
- Contains **6 missing values**.
- Missing values should be imputed (mean/median) or the affected rows removed.

### customer_rating
- Contains **8 missing values**.
- Missing values should be imputed before modeling.

### holiday_flag
- Already binary (0/1).
- No transformation required.

### store_id
- Acts as a unique identifier.
- Not suitable as a numerical predictor.

---

# 6. Variables That May Not Be Useful for Regression

| Variable | Reason |
|----------------|------------------------------------------------|
| store_id | Unique identifier; does not provide predictive information |
| monthly_profit | Not recommended when predicting monthly_sales because it is a business outcome closely related to sales and may introduce target leakage |
